#### 关系

基本概念：有序对的集合	（从A到B、A上）

表示：集合、矩阵、关系图

运算：合成、逆

性质：自反、反自反、对称、反对称、传递（在A上）	5个性质

闭包：自反	R∪IA

​			对称	R∪R逆（R^-1）

​			传递	R∪R^2∪R^3∪......

​			特例：|A| = n，R∪R^2∪...∪R^n

Warshall算法：把第k行加到第k列中“1”所在行
